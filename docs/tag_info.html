标签：Info.*

Version (4.00)

说明：

站点信息的调用。

显示元素：

名称 描述
Info.SiteName 显示站点名称
Info.SiteUrl 显示站点地址
Info.SiteMail 显示站点联系E-Mail
Info.Version 显示系统版本号
Info.ThemeName 显示当前风格名称
Info.ColumnTotal 显示站点栏目总数
Info.TopicTotal 显示站点文章总数
Info.UserTotal 显示站点注册用户总数
Info.CommentTotal 显示站点评论总数

调用范例：


<div>
 <ul>
  <li>栏目总数：<!--Info.ColumnTotal--></li>
  <li>文章总数：<!--Info.TopicTotal--></li>
  <li>会员总数：<!--Info.UserTotal--></li>
  <li>评论总数：<!--Info.CommentTotal--></li>
 </ul>
</div>
