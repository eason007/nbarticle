标签：Column.List

Version (4.00)

说明：

显示某栏目下的子栏目。

调用方法：

参数 描述
Parameter(ID) ID - 父栏目ID。

显示元素：

名称 描述
Title 栏目名称
Url 连接地址
Info 栏目简介
Total 文章总数
调用范例：

<div>
 <ul>
  <!--Column.List Begin-->
  <!--Parameter(1)-->
   <li><a href="<!--Url-->" target="_blank" title="<!--Info-->"><!--Name-->[<!--Total-->]</a></li>
  <!--Column.List End-->
 </ul>
</div>
