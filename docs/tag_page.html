标签：Page.*

Version (4.00)

说明：

页面基本元素的调用。

显示元素：

名称 描述
Page.Head 调用头部模块
Page.Foot 调用底部模块
Page.CSS 调用CSS模块
Page.Nav 显示页面的当前位置
Page.Title 页面标题
Page.Description 页面简介
Page.Keyword 页面关键词
Page.Path 网站路径

调用范例：


<html>
 <head>
  <title><!--Page.Title--></title>
  <meta name="keywords" content="<!--Page.Keyword-->">
 </head>
 <body>
  <div>当前位置：<!--Page.Nav--></div>
 </body>
</html>
