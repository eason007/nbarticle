标签：Placard.List

Version (4.00)

说明：

站点公告的调用。

调用方法：

调用方法 描述
Parameter(Top) 调用指定数目的公告。按发布时间倒序排列。
Parameter(ID=) 调用指定ID的公告信息

显示元素：

名称 描述
Title 公告标题
Content 公告内容
AddTime 发布时间
OverTime 过期时间
调用范例1：

<div>
 <ul>
  <!--Placard.List Begin-->
  <!--Parameter(2)-->
   <li>标题：<!--Title--></li>
  <!--Placard.List End-->
 </ul>
</div>

调用范例2：

<div>
 <ul>
  <!--Placard.List Begin-->
  <!--Parameter(ID=2)-->
   <li>标题：<!--Title-->[<!--AddTime-->]</li>
  <!--Placard.List End-->
 </ul>
</div>
