标签：Comment.List

Version (4.00)

说明：

提取评论。

调用方法：

调用参数 描述
Parameter(Top, ArticleID, StrLen) Top - 提取的评论数。
ArticleID - 指定提取某文章下的评论。0则为全站所有评论。
StrLen - 提取评论内容的字数。

显示元素：

名称 描述
UserName 评论用户
Content 评论内容
Date 发布时间
ArticleUrl 文章连接
ArticleTitle 文章标题
调用范例：

<div>
 <!--Comment.List Begin-->
 <!--Parameter(10,0,150)-->
  <div>
   <!--UserName-->说：
   <p><!--Content--></p>
  </div>
 <!--Comment.List End-->
</div>
