标签：Topic.List

Version (4.00)

说明：

文章的调用。

调用方法：

参数 描述
Parameter(Top, ColumnID, Type, TitleLen, IsChild) Top - 提取的文章总数。
ColumnID - 提取指定栏目ID的文章，如设置为0，则提取所有栏目的文章。
Type - 0为最新文章；1为推荐文章；3为图片文章；4为浏览最多文章；5为随机提取。
TitleLen - 文章显示的标题长度。
IsChild - 是否包含指定栏目的子栏目。1为包含；0为不包含。只有ColumnID不为0时才有效。

显示元素：

名称 描述
Title 文章标题
Url 文章连接地址
SubTitle 副标题。如设置了副标题的连接，则自动加上连接。
Img 文章头图
Date 发布日期
Time 发布时间
Author 作者
Icon 文章小图标
ColumnName 所属栏目名称
ColumnUrl 所属栏目连接
调用范例：

<div>
 <ul>
  <!--Topic.List Begin-->
  <!--Parameter(10,0,1,23,0)-->
   <li><a href="<!--Url-->" target="_blank"><!--Title--></a><!--Date--></li>
  <!--Topic.List End-->
 </ul>
</div>
