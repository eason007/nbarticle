标签：List.*

Version (4.00)

说明：

列表页的内容调用。

显示元素：

名称 描述
List.ID 栏目ID
List.Name 栏目名称
List.Description 栏目简介
List.TopicTotal 栏目文章总数
List.Topic 文章循环块
List.PageNav 分页导航

List.Topic内显示元素：

名称 描述
Title 文章标题
Url 连接地址
SubUrl 副标题连接
SubTitle 副标题
Date 发布日期
Time 发布时间
CommentNum 评论总数
Summary 摘要
LastComment 最新评论内容
ViewNum 浏览总数
Icon 小图标
Img 文章头图
Author 作者
Tag 标签
调用范例：

<div>
 <span><!--List.Name--></span>
 <ul>
  <!--List.Topic Begin-->
   <li><a href="<!--Url-->" target="_blank"><!--Title--></a><!--Date--></li>
  <!--List.Topic End-->
 </ul>
 <span><!--List.PageNav--></span>
</div>
