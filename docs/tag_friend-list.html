标签：Friend.List

Version (4.00)

说明：

友情连接的调用。

调用方法：

参数 描述
Parameter(Top, ColumnID, Style) Top - 提取的友情连接总数。
ColumnID - 友情连接所在的栏目ID，如设置为0，则提取首页的友情连接。
Style - 1为图片连接，0为文本连接。

显示元素：

名称 描述
Name 站点名称
Url 连接地址
Img 站点图片
Info 站点简介
调用范例：

<div>
 <ul>
  <!--Friend.List Begin-->
  <!--Parameter(10,0,1)-->
   <li><a href="<!--Url-->" target="_blank" title="<!--Name-->"><img src="<!--Img-->" /></a></li>
  <!--Friend.List End-->
 </ul>
</div>
